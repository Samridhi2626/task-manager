# Task Manager App

A full-stack Task Manager application built using React, Node.js, Express.js, MongoDB, and Tailwind CSS.

## Features

- User Registration and Login
- Create Tasks
- Update Task Status
- Delete Tasks
- Todo / In Progress / Done workflow

## Tech Stack

Frontend:
- React
- Tailwind CSS
- Axios

Backend:
- Node.js
- Express.js
- MongoDB
- JWT Authentication
